# BubbleSort
Repositório para o projeto final da disciplina de Lógica Computacional 1 - UnB

## Estrutura do Projeto:
- `Projeto1_BubbleSort.v`: Codigo fonte em Coq contendo a implementacao do algoritmo.
- `relatorio.pdf`: Relatório do projeto em formato PDF.

## Pre-requisitos:
Para compilar o projeto é necessário:
- `Coq` (versao 8.10 ou superior recomendada)
- `CoqIDE` (versao 8.10 ou superior recomendada)

## Execução:

Para verificar as provas do arquivo Coq, execute o seguinte comando no terminal:

```
coqc Projeto1_BubbleSort.v
```
